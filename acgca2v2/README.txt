Applied Cryptography CA2

Instructions:

1) Download assignment folder and unzip!
2) To run the program, open two command prompts.
3) Compile the server and client code using javac (E.g. javac server.java)
4) Run the server program FIRST, then the client program SECOND. (E.g java server)

Additional notes:

1) Server will continue to listen for new client connections until terminated manually.
2) When server asks "Do you want to download securely?" , picking no will just use the normal unsecured FTP program.
